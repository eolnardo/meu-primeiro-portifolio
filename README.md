# Portfólio Pessoal - Estudos Web

Este é um projeto de portfólio estático desenvolvido para consolidar conhecimentos básicos de **HTML5** e **CSS3**. 

O projeto faz parte do meu cronograma de revisão técnica enquanto curso **Redes de Computadores**, unindo o desenvolvimento de software com a compreensão da infraestrutura web.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Estrutura)
* CSS3 (Estilização e Flexbox)
* Git/GitHub (Controle de versão)
